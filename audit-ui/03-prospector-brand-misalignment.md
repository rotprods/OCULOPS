# BRAND MISALIGNMENT: OCULOPS vs CURRENT

## 1. Target Style

Apple × Higgsfield × Whisk × Pomelli

- **Boutique premium:** High-end, expensive feeling.
- **Editorial minimal:** Typography-driven, generous negative space, fewer boxes.
- **Yellow/white/black:** A disciplined, monochromatic palette punctuated by yellow for action/intelligence.

## 2. Current Misalignments in Prospector Hub

1. **Color Palette is too generic:** Extensive use of standard web colors (green `#34A853`, blue `#4285F4`, red `#E4405F`). This destroys the "yellow/white/black intelligence instrument" mandate.
2. **Iconography is cheap:** Emojis (`📍`, `⚙️`, `🧠`) are used universally. These must be replaced with sharp, minimal SVGs.
3. **Containers are SaaS-standard:** The `.card` with `border: 1px solid var(--border-subtle)` is classic generic SaaS. We need glass surfaces, invisible containers defined by typography, or subtle gradients.
4. **Agent Visualization is absent:** CORTEX, HUNTER, and ATLAS are running in the background, but the UI never shows them "living". The user doesn't feel like they are commanding AI; they feel like they are clicking buttons in a database.
5. **Typography is functional but not editorial:** Inter is used globally as a functional sans, but there is no editorial pairing (like Times New Roman / Canela) to give it that "Whisk/Pomelli" boutique aesthetic.
